## Context

Our admin console journeys allow for viewing a table of teachers, which can be searched and filtered. As well as this, an admin can also view teachers by school. This is by searching for the school and then viewing the 'teachers' tab within the school's details.

We'd noted that admin users had requested the ability to be able to search and filter teachers at a school.

This is because it can often be the case that there have been many teachers registered at a school, especially as links to teacher details remain from previous contract periods.

## What we've designed

From discussing with admin users, we've identified that searching by name, TRN and API participant ID are the most common ways to search for a teacher.

We've also identified that filtering by role and contract period are particularly useful.

![A screenshot of the journey where an admin is able to search and filter teachers at a school.](/ecf-v2/searching-and-filtering-in-admin-console/image.png)

We've designed the 'teachers' tab to have these search and filter options, and also updated the overall 'teachers' list (not at a school level) page to also include these options.
